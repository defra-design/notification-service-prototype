Import of products, animals, food and feed system (IPAFFS)

Importers and import agents need to use IPAFFS to notify authorities about imports to Great Britain.

Register for IPAFFS

Users must sign in to access the service. Those already using other Defra services can register for IPAFFS by selecting ‘register for more services’. Import agents acting on behalf of importers must register using their own business details and should not select ‘intermediaries’ on the ‘Manage account’ screen.

When you raise a notification in IPAFFS, you are responsible for the consignment from when it enters Great Britain until the authorities complete their checks.

Create an import notification

Import documents must be in electronic format (for example PDF). Great Britain accepts verifiable PDF GB health certificates from the EU, Iceland, Liechtenstein and Norway for live animals and animal products.

Documents must be uploaded to the import notification before submission.
